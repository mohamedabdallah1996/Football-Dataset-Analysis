Introduction¶

The ultimate Soccer SQLite database has data about 25k+ matches, players & teams attributes for European Professional Football from season 2008/2009 to 2016/2017.

Dataset Description

  • +25,000 matches
  • +10,000 players
  • 11 European Countries with their lead championship
  • Seasons 2008 to 2016
  • Players and Teams' attributes* sourced from EA Sports' FIFA video game series, including the weekly updates
  • Team line up with squad formation (X, Y coordinates)
  • Betting odds from up to 10 providers
  • Detailed match events (goal types, possession, corner, cross, fouls, cards etc) for +10,000 matches

If you want to access or download this dataset, you can get more details here.

In [10]:
matches = pd.read_sql("""
                      select Match.id, 
                             Country.name as country_name, 
                             League.name as league_name, 
                             season,
                             stage, 
                             date, 
                             HT.team_long_name as home_team, 
                             AT.team_long_name as away_team,
                             home_team_goal,
                             away_team_goal,
                             home_team_goal + away_team_goal as total_goals,
                             case
                                 when home_team_goal > away_team_goal then 1
                                 else 0
                             end as home_team_win
                             
                      from Match
                      join Country on Match.country_id = Country.id
                      join League on Match.league_id = League.id
                      join Team as HT on Match.home_team_api_id = HT.team_api_id 
                      join Team as AT on Match.away_team_api_id = AT.team_api_id
                      order by date
                      """, conn)

Tip: Note that Team is joined twice. This is a tricky one, as while we are using the same table name, we basically bring two different copies (and rename them using AS). leagues_analysis = pd.read_sql("""
                        select Match.id, 
                             Country.name as country_name, 
                             League.name as league_name, 
                             count(distinct stage) as number_of_stages,
                             count(distinct HT.team_long_name) as number_of_teams,
                             count(HT.team_long_name) as number_of_matches
                      from Match
                      join Country on Match.country_id = Country.id
                      join League on Match.league_id = League.id
                      join Team as HT on Match.home_team_api_id = HT.team_api_id 
                      join Team as AT on Match.away_team_api_id = AT.team_api_id
                      group by league_name
                      having count(distinct stage) > 10
                      order by country_name 
                    """, conn)

def home_team_result(row):
    if row['home_team_goal'] > row['away_team_goal']:
        return 'win'
    elif row['home_team_goal'] < row['away_team_goal']:
        return 'lose'
    else:
        return 'draw'
    

def away_team_result(row):
    if row['home_team_goal'] > row['away_team_goal']:
        return 'lose'
    elif row['home_team_goal'] < row['away_team_goal']:
        return 'win'

Research Question 7 (What are the top winner teams per each league?)¶

To answer this question, we need to add new column which has the name of the winner team per match. so we can group the dataframe by the league_name and count for every team number of wins.

# add winner column
def winner(row):
    if row['home_team_goal'] > row['away_team_goal']:
        return row['home_team']
    elif row['home_team_goal'] < row['away_team_goal']:
        return row['away_team']
    else:
        return 'draw'
    
matches['winner'] = matches.apply(lambda row: winner(row), axis=1)
